# Salon Site — Документация проекта

Это корневой индекс документации проекта **сайта салона**.

> **Для любого чата / ассистента:** прочитай файлы в этой папке (`docs/`) — в них хранится вся история и весь контекст проекта. После этого ты будешь знать всё то же, что и предыдущий чат. **Всегда дополняй эти файлы** по ходу работы.

## Файлы

| Файл | Что внутри |
|------|------------|
| [README.md](README.md) | Этот индекс — с чего начать |
| [CONTEXT.md](CONTEXT.md) | Полный контекст проекта: цель, решения, состояние |
| [HISTORY.md](HISTORY.md) | Хронология чатов и сделанного (лог по датам) |

## Правила ведения docs

1. **Каждый чат**, начиная работу, сначала читает `CONTEXT.md` и `HISTORY.md`.
2. **Каждый чат**, заканчивая работу, дополняет:
   - `HISTORY.md` — новой записью с датой: что обсудили и что сделали.
   - `CONTEXT.md` — если изменились цель, решения или состояние проекта.
3. Пиши кратко и по делу, чтобы следующий чат быстро вошёл в курс.
4. Не удаляй старые записи истории — только добавляй новые.
