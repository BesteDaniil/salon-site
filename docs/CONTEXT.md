## 1. О проекте

- **Название:** Сайт салона (рабочее).
- **Что это:** веб-сайт для салона.
- **Расположение проекта:** `D:\salon-site`
- **Документация:** `D:\salon-site\docs`

## 2. Цель

- Создать сайт **салона красоты** в **тёмно-зелёных тонах**.
- Дизайн-ориентир: чужой сайт «BEAUTY ROOM» (Behance) — берём только структуру/дух, **не копируем** (название, тексты, фото — свои).
- Детали салона (реальное название, адрес, контакты, услуги, команда, фото) — **уточнить у владельца**.

## 4. Технологии / стек

- **HTML + CSS + JS, без сборки** (статический сайт). Шрифты: Cormorant Garamond + Jost (Google Fonts).
- Причина: открывается двойным кликом, легко править владельцу и хостить где угодно.

## 5. Текущее состояние

- Свёрстаны 2 страницы: `index.html` (главная) и `about.html` («О нас») в тёмно-зелёной теме.
- Стили — `css/styles.css`, скрипты — `js/main.js`.
- Фото — заглушки-градиенты; инструкция замены в `images/README.md`.
- Бренд/тексты/адрес сейчас взяты из референса как **плейсхолдеры** — ждём реальные данные.
