# Картинки

Сейчас на сайте вместо фото — тёмно-зелёные градиенты-заглушки (заданы в `css/styles.css`
в блоке `:root` как переменные `--img-*`).

## Как поставить реальные фото

1. Положи сюда файлы, например: `hero.jpg`, `makeup.jpg`, `color.jpg` и т.д.
2. В `css/styles.css` замени значение нужной переменной. Было:
   ```css
   --img-hero: linear-gradient(135deg, #20342a, #0e1a14);
   ```
   Стало:
   ```css
   --img-hero: url("../images/hero.jpg");
   ```

## Список переменных

- `--img-hero` — большое фото на главной (герой)
- `--img-manicure`, `--img-color` — мини-карточки в герое
- `--img-makeup`, `--img-color`, `--img-nails`, `--img-brows`, `--img-haircut`, `--img-tattoo`, `--img-piercing` — плитки «Услуги»
- `--img-care` — карточка акции «Заботливый уход»
- `--img-face` — фото в подвале
- `--img-interior`, `--img-interior2..5` — фото салона на странице «О нас»
- `--img-master1..4` — фото мастеров в блоке «Команда»

Рекомендуемый размер фото: 1200–1600 px по ширине, формат `.jpg` или `.webp`.
